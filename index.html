Php-htmldiff

A library for comparing two HTML files/snippets and highlighting the differences using simple HTML.

php-htmldiff

This HTML Diff implementation is a PHP port of the ruby implementation found at https://github.com/myobie/htmldiff. This is also available in C# at https://github.com/Rohland/htmldiff.net.
